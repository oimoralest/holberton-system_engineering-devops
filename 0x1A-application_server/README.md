# 0x1A. Application server
## Description
    This project is about to configure an app server
## Requirements
- Ubuntu 16.04 LTS
- Python 3.x => 3.5
- Flask 1.1.2
- sqlalchemy 1.3.23
- MySQLdb 1.3.10
- Flask-cors 3.0.10
- gunicorn 20.0.4
